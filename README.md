# Weather App

A live weather application that fetches real-time data using the OpenWeatherMap API.

## Features
- Search current weather by city name
- Displays temperature, feels like, high and low
- Shows current weather conditions
- Clear button to reset the display
- Input validation and error handling for invalid cities

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Use
1. Enter a city name in the search bar
2. Click Search
3. View live weather data for that city

## Setup
This project requires an OpenWeatherMap API key.
1. Sign up at openweathermap.org
2. Create a config.js file with your API key
3. Open weather-app.html in your browser
